# Financial Analysis & Banking Analytics using Python

## Project Overview

This project focuses on financial analysis and banking analytics using Python on Central Bank of India and Union Bank datasets.

The project includes:
- Financial Modelling
- Regression Analysis
- ANOVA
- Correlation Analysis
- Bibliometric Analysis
- Data Visualization
- CSV Data Processing
- Banking Analytics

# Major Tasks

## 1. Bibliometric Analysis using VOSviewer
Performed thematic analysis on banking research and financial datasets.

## 2. CSV Import and Data Cleaning
Imported banking datasets and cleaned financial variables using Pandas.

## 3. Financial Visualizations
Created bar charts, correlation heatmaps and trend analysis.

## 4. Descriptive Statistics
Generated summary statistics for banking financial indicators.

## 5. Correlation Analysis
Analyzed relationships between:
- Equity Capital
- Reserves
- Deposits
- Total Assets

## 6. Regression Analysis
Built OLS regression models using financial indicators.

## 7. Simultaneous Equation Model
Implemented financial relationship models using Python.

## 8. ANOVA Analysis
Performed One-Way ANOVA for statistical testing.

## 9. Financial Calculator
Created Python-based financial calculator using conditional statements.

## 10. Financial Modelling
Built financial modelling system using CSV datasets and Python automation.
